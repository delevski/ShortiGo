# ShortiGo v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Flutter (iOS + Android) short-drama streaming app where videos play instantly on swipe, content is series-based, and revenue flows from VIP subscriptions + rewarded ads.

**Architecture:** Clean-Architecture-lite with 4 layers (Presentation → Application → Domain → Data). Riverpod for state, go_router for routing, Firebase for backend, better_player for video, AdMob for ads, RevenueCat for IAP.

**Tech Stack:** Flutter 3.x, Dart 3.x, Riverpod 2 (codegen), go_router, firebase_auth, cloud_firestore, firebase_storage, google_mobile_ads, purchases_flutter (RevenueCat), better_player, drift (SQLite), sentry_flutter, firebase_performance, freezed, json_serializable, Cloud Functions (Node 20 + TypeScript).

**Spec:** `docs/superpowers/specs/2026-06-01-shortigo-design.md`

## Working assumptions

- The executing engineer is a skilled developer but new to this specific stack (Flutter + Firebase + Riverpod + freezed). Each step shows exact commands and complete code.
- macOS host (per project env), Xcode for iOS, Android Studio for Android. CI is not in v1 scope.
- Flutter SDK is installed and `flutter doctor` is green. The plan does not install Flutter.
- Firebase CLI (`firebase`) is installed and authenticated.
- Node 20+ and npm are installed (for Cloud Functions).
- The user provides a Firebase project ID for `dev` and `prod` flavors.

## Plan layout

This plan is organized as **7 milestones**, one per the rollout in spec §13. Each milestone is its own chapter and can be executed as a unit. Tasks within a milestone are sequential and small (2–5 minutes per step).

- **Milestone 1 — Foundations** (Tasks 1–11): repo, Flutter scaffold, theme, env, router, auth, bottom nav, build flavors
- **Milestone 2 — Discover** (Tasks 12–22): data model, Firestore layer, drift cache, /discover page, /series detail
- **Milestone 3 — Player + Shorts** (Tasks 23–36): single-episode player, vertical feed, pre-cache manager, performance
- **Milestone 4 — Rewards + Wallet** (Tasks 37–50): daily check-in, AdMob integration, Cloud Functions, wallet UI
- **Milestone 5 — Subscriptions** (Tasks 51–59): RevenueCat IAP, VIP gate, server-side validation
- **Milestone 6 — Hardening** (Tasks 60–70): integration tests, performance, error UX
- **Milestone 7 — Release** (Tasks 71–75): submission prep, monitoring

```dart
import 'package:flutter/foundation.dart';

enum AppFlavor { dev, prod }

class Env {
  Env._({
    required this.flavor,
    required this.firebaseProjectId,
    required this.sentryDsn,
    required this.adMobAppIdIos,
    required this.adMobAppIdAndroid,
    required this.adMobRewardedUnitIdIos,
    required this.adMobRewardedUnitIdAndroid,
    required this.revenueCatApiKeyIos,
    required this.revenueCatApiKeyAndroid,
  });

  final AppFlavor flavor;
  final String firebaseProjectId;
  final String sentryDsn;
  final String adMobAppIdIos;
  final String adMobAppIdAndroid;
  final String adMobRewardedUnitIdIos;
  final String adMobRewardedUnitIdAndroid;
  final String revenueCatApiKeyIos;
  final String revenueCatApiKeyAndroid;

  bool get isProd => flavor == AppFlavor.prod;

  /// Built from --dart-define values. Defaults to dev if unset.
  factory Env.fromDefines() {
    const flavorStr = String.fromEnvironment('ENV', defaultValue: 'dev');
    final flavor = flavorStr == 'prod' ? AppFlavor.prod : AppFlavor.dev;

    return Env._(
      flavor: flavor,
      firebaseProjectId: const String.fromEnvironment(
        'FIREBASE_PROJECT_ID',
        defaultValue: 'shortigo-dev',
      ),
      sentryDsn: const String.fromEnvironment(
        'SENTRY_DSN',
        defaultValue: '',
      ),
      adMobAppIdIos: const String.fromEnvironment(
        'ADMOB_APP_ID_IOS',
        defaultValue: 'ca-app-pub-3940256099942544~1458002511',
      ),
      adMobAppIdAndroid: const String.fromEnvironment(
        'ADMOB_APP_ID_ANDROID',
        defaultValue: 'ca-app-pub-3940256099942544~3347511713',
      ),
      adMobRewardedUnitIdIos: const String.fromEnvironment(
        'ADMOB_REWARDED_IOS',
        defaultValue: 'ca-app-pub-3940256099942544/1712485313',
      ),
      adMobRewardedUnitIdAndroid: const String.fromEnvironment(
        'ADMOB_REWARDED_ANDROID',
        defaultValue: 'ca-app-pub-3940256099942544/5224354917',
      ),
      revenueCatApiKeyIos: const String.fromEnvironment(
        'RC_API_KEY_IOS',
        defaultValue: '',
      ),
      revenueCatApiKeyAndroid: const String.fromEnvironment(

  }

  @override
  String toString() =>
      'Env(flavor: ${flavor.name}, project: $firebaseProjectId)';

  @override

      other is Env &&
      other.flavor == flavor &&
      other.firebaseProjectId == firebaseProjectId;

  int get hashCode => Object.hash(flavor, firebaseProjectId);
}
```

```dart
FriendlyError friendlyErrorFor(Object e) {
  final s = e.toString();
  if (s.contains('SocketException') || s.contains('Failed host lookup')) {
```

```dart
import 'package:cloud_firestore/cloud_firestore.dart';

import '../../domain/entities/category.dart';
import '../../domain/entities/series.dart';
import 'series_repository.dart';

class FirestoreSeriesRepository implements SeriesRepository {
  FirestoreSeriesRepository(this._db, {required this.featuredDocId});

  final FirebaseFirestore _db;
  final String featuredDocId;

  CollectionReference<Map<String, dynamic>> get _col =>
      _db.collection('series');

  @override
  Future<List<Series>> forYou({int limit = 20}) async {
    final snap = await _db.collection('admin').doc(featuredDocId).get();
    final data = snap.data();
    if (data == null) return [];
    final ids = List<String>.from(data['seriesIds'] ?? const <String>[]);
    if (ids.isEmpty) return [];
    final docs = await _db.collection('series').where(FieldPath.documentId, whereIn: ids).get();
    final byId = {for (final d in docs.docs) d.id: d};
    final ordered = [for (final id in ids) if (byId[id] != null) byId[id]!];
    return ordered
        .take(limit)
        .map((d) => _toSeries(d))
        .where((s) => s.isPublished)
        .toList();
  }

  @override
  Future<List<Series>> byCategory(Category category, {int limit = 20}) async {
    if (category == Category.forYou) return forYou(limit: limit);
    Query<Map<String, dynamic>> q = _col.where('isPublished', isEqualTo: true);
    if (category == Category.vip) {
      q = q.where('isVip', isEqualTo: true);
    } else {
      q = q.where('category', isEqualTo: category.id);
    }
    if (category == Category.hot) {
      q = q.orderBy('popularity', descending: true);
    } else {
      q = q.orderBy('createdAt', descending: true);
    }
    final snap = await q.limit(limit).get();
    return snap.docs.map(_toSeries).toList();
  }

  @override
  Future<Series> byId(String id) async {
    final d = await _col.doc(id).get();
    return _toSeries(d);
  }

  Series _toSeries(QueryDocumentSnapshot<Map<String, dynamic>> d) {
    return Series.fromJson({...d.data(), 'id': d.id});
  }
}
```

```dart
import 'package:better_player/better_player.dart';

import '../../../domain/entities/episode.dart';

class VideoPreCacheManager {
  VideoPreCacheManager({this.windowSize = 3});

  int windowSize;

  /// episode.id -> controller. Holds up to `windowSize` entries.
  final Map<String, BetterPlayerController> _controllers = {};
  final Map<String, String> _urls = {};
  String? _currentId;

  Future<void> setupAt({
    required int currentIndex,
    required List<Episode> episodes,
    required Future<String> Function(Episode) urlFor,
  }) async {
    _currentId = episodes[currentIndex].id;

    final keepIds = <String>{};
    for (var offset = -1; offset <= 1; offset++) {
      final i = currentIndex + offset;
      if (i < 0 || i >= episodes.length) continue;
      keepIds.add(episodes[i].id);
    }
    // Also pre-init the +2 (next-of-next) when possible
    final next = currentIndex + 2;
    if (next < episodes.length) keepIds.add(episodes[next].id);

    // Drop controllers outside the keep set
    final toRemove = _controllers.keys.where((id) => !keepIds.contains(id)).toList();
    for (final id in toRemove) {
      await _controllers[id]?.dispose();
      _controllers.remove(id);
      _urls.remove(id);
    }

    // Init missing ones
    for (final id in keepIds) {
      if (_controllers.containsKey(id)) continue;
      final ep = episodes.firstWhere((e) => e.id == id);
      try {
        final url = await urlFor(ep);
        _urls[id] = url;
        final controller = BetterPlayerController(
          const BetterPlayerConfiguration(
            autoPlay: false,
            aspectRatio: 9 / 16,
            fit: BoxFit.cover,
            controlsConfiguration: BetterPlayerControlsConfiguration(showControls: false),
          ),
        );
        await controller.setupDataSource(BetterPlayerDataSource.network(
          url,
          cacheConfiguration: const BetterPlayerCacheConfiguration(useCache: true),
        ));
        _controllers[id] = controller;
      } catch (_) {
        // best-effort pre-cache; failures don't block the feed
      }
    }
  }

  BetterPlayerController? controllerFor(String id) => _controllers[id];

  Future<void> play(String id) async {
    for (final c in _controllers.values) {
      await c.pause();
    }
    await _controllers[id]?.play();
  }

  Future<void> pauseAll() async {
    for (final c in _controllers.values) {
      await c.pause();
    }
  }

  Future<void> disposeAll() async {
    for (final c in _controllers.values) {
      await c.dispose();
    }
    _controllers.clear();
    _urls.clear();
  }
}
```

```dart
  if (!Platform.isAndroid) return;
```

```ts
import * as admin from 'firebase-admin';

const db = admin.firestore();

export type TxType = 'adReward' | 'purchase' | 'spend' | 'refund';

export interface LedgerEntry {
  userId: string;
  type: TxType;
  coinsDelta: number;
  bonusDelta: number;
  reference?: string;
}

export async function appendLedgerAndApply(entry: LedgerEntry): Promise<void> {
  const userRef = db.collection('users').doc(entry.userId);
  const txRef = userRef.collection('transactions').doc();

  const batch = db.batch();
  batch.set(txRef, {
    ...entry,
    id: txRef.id,
    at: admin.firestore.FieldValue.serverTimestamp(),
  });
  if (entry.coinsDelta !== 0) {
    batch.update(userRef, { coins: admin.firestore.FieldValue.increment(entry.coinsDelta) });
  }
  if (entry.bonusDelta !== 0) {
    batch.update(userRef, { bonus: admin.firestore.FieldValue.increment(entry.bonusDelta) });
  }
  await batch.commit();
}
```

```ts
import * as admin from 'firebase-admin';
import { appendLedgerAndApply } from './lib/ledger';

admin.initializeApp();

const DAILY_CAP = 50;
const REWARD_AMOUNT = 12;

export const grantAdReward = functions.https.onCall(async (data, context) => {
  if (!context.auth) {
    throw new functions.https.HttpsError('unauthenticated', 'Sign in required');
  }
  const userId = context.auth.uid;
  const { adUnitId, adId } = data as { adUnitId: string; adId: string };
  if (typeof adUnitId !== 'string' || typeof adId !== 'string') {
    throw new functions.https.HttpsError('invalid-argument', 'Missing adUnitId or adId');
  }

  const since = admin.firestore.Timestamp.fromMillis(Date.now() - 24 * 60 * 60 * 1000);
  const todayTx = await admin
    .firestore()
    .collection('users')
    .doc(userId)
    .collection('transactions')
    .where('type', '==', 'adReward')
    .where('at', '>=', since)
    .get();

  if (todayTx.size >= DAILY_CAP) {
    throw new functions.https.HttpsError('resource-exhausted', 'Daily ad cap reached');
  }

  await appendLedgerAndApply({
    userId,
    type: 'adReward',
    coinsDelta: 0,
    bonusDelta: REWARD_AMOUNT,
    reference: `${adUnitId}:${adId}`,
  });
  return { ok: true, bonusDelta: REWARD_AMOUNT };
```

```dart
    if (_initialized) return;
    if (appleApiKey.isEmpty && googleApiKey.isEmpty) return; // dev with no keys
    final key = Platform.isIOS ? appleApiKey : googleApiKey;

    return offerings.all.values
        .map((o) => IapOffering(
              identifier: o.identifier,
              packages: o.availablePackages
                  .map((p) => IapPackage(identifier: p.identifier, priceString: p.storeProduct.priceString))
                  .toList(),
            ))
        .toList();

        if (p.identifier == packageIdentifier) {

          break;
        }

      if (pkg != null) break;

    if (pkg == null) return false;
    try {
```

```dart
import 'package:firebase_auth/firebase_auth.dart' as fb;
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'package:google_sign_in/google_sign_in.dart';

import '../../../core/providers.dart';

class AuthState {
  const AuthState({this.user, this.isLoading = false, this.error});
  final fb.User? user;
  final bool isLoading;
  final String? error;
}

class AuthNotifier extends AsyncNotifier<AuthState> {
  @override
  Future<AuthState> build() async {
    final auth = ref.watch(currentAuthUserProvider).value;
    return AuthState(user: auth);
  }

  Future<void> signInWithEmail(String email, String password) async {
    state = const AsyncLoading<AuthState>().copyWithPrevious(state);
    try {
      final cred = await fb.FirebaseAuth.instance.signInWithEmailAndPassword(
        email: email, password: password,
      );
      state = AsyncData(AuthState(user: cred.user));
    } catch (e) {
      state = AsyncData(AuthState(error: e.toString()));
    }
  }

  Future<void> registerWithEmail(String email, String password) async {
    state = const AsyncLoading<AuthState>().copyWithPrevious(state);
    try {
      final cred = await fb.FirebaseAuth.instance.createUserWithEmailAndPassword(
        email: email, password: password,
      );
      await cred.user?.sendEmailVerification();
      state = AsyncData(AuthState(user: cred.user));
    } catch (e) {
      state = AsyncData(AuthState(error: e.toString()));
    }
  }

  Future<void> signInWithGoogle() async {
    state = const AsyncLoading<AuthState>().copyWithPrevious(state);
    try {
      final googleUser = await GoogleSignIn().signIn();
      if (googleUser == null) {
        state = AsyncData(AuthState(user: ref.read(currentAuthUserProvider).value));
        return;
      }
      final googleAuth = await googleUser.authentication;
      final credential = fb.GoogleAuthProvider.credential(
        accessToken: googleAuth.accessToken,
        idToken: googleAuth.idToken,
      );
      final cred = await fb.FirebaseAuth.instance.signInWithCredential(credential);
      state = AsyncData(AuthState(user: cred.user));
    } catch (e) {
      state = AsyncData(AuthState(error: e.toString()));
    }
  }

  Future<void> signOut() async {
    await fb.FirebaseAuth.instance.signOut();
    state = const AsyncData(AuthState());
  }
}

final authNotifierProvider =
    AsyncNotifierProvider<AuthNotifier, AuthState>(AuthNotifier.new);
```

```dart
import 'package:firebase_core/firebase_core.dart';
import 'package:cloud_firestore/cloud_firestore.dart';
import 'package:firebase_storage/firebase_storage.dart';
import 'dart:io';

import '../shortigo/lib/bootstrap/firebase_options_dev.dart' as dev;

Future<void> main(List<String> args) async {
  if (args.length < 4) {
    print('Usage: dart run tools/upload_episode.dart <seriesId> <order> <videoPath> <thumbPath> [isVip]');
    exit(1);
  }
  final seriesId = args[0];
  final order = int.parse(args[1]);
  final videoPath = args[2];
  final thumbPath = args[3];
  final isVip = args.length > 4 && args[4] == 'true';

  await Firebase.initializeApp(options: dev.DefaultFirebaseOptions.currentPlatform);
  final db = FirebaseFirestore.instance;
  final storage = FirebaseStorage.instance;

  final videoFile = File(videoPath);
  final thumbFile = File(thumbPath);

  final videoRef = storage.ref('series/$seriesId/episodes/${seriesId}_e$order.mp4');
  final thumbRef = storage.ref('series/$seriesId/thumbnails/${seriesId}_e$order.jpg');

  print('Uploading video...');
  await videoRef.putFile(videoFile, SettableMetadata(cacheControl: 'public, max-age=86400'));
  print('Uploading thumbnail...');
  await thumbRef.putFile(thumbFile, SettableMetadata(cacheControl: 'public, max-age=86400'));

  final videoUrl = await videoRef.getDownloadURL();
  final thumbUrl = await thumbRef.getDownloadURL();

  await db.collection('episodes').doc('${seriesId}_e$order').set({
    'seriesId': seriesId,
    'order': order,
    'videoUrl': videoUrl,
    'thumbnailUrl': thumbUrl,
    'durationSec': 60,
    'isVipLocked': isVip,
  });

  print('Episode $order uploaded to series $seriesId.');
}
```
