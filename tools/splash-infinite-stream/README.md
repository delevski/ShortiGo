# Vortex splash (HyperFrames)

3-second portrait splash based on the ShortiGo hero artwork: spinning light ring, glowing play icon with internal wave motion, and flinging title copy.

## Timeline

| Time | Beat |
|------|------|
| 0.0–0.4s | Ring fades in and begins continuous spin |
| 0.15–3.0s | Play icon glow pulse + internal wave lines |
| 0.84–1.4s | **SHORTIGO** flings up with overshoot |
| 1.08–1.6s | Tagline flings in |

## Assets

- Reference artwork: `assets/branding/splash_hero.png`
- Flutter runtime: `lib/features/splash/presentation/shortigo_vortex_splash.dart`

## Commands

```bash
cd tools/splash-infinite-stream
npm run dev
npm run check
npm run render
```
